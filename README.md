## ✨ Не береді?

- **Жеке сілтеме** — әр тапсырыс өз URL-мен ашылады
- **Музыка** — дайын қазақ музыкасынан таңдау немесе өз файлын жүктеу
- **Фото** — кейіпкерлердің суреттерімен жекелендіру
- **Үш тілде** — Қазақша / O'zbekcha / Русский
- **2ГИС сілтемесі** — тойхана орнын картадан тікелей ашу

## 🔄 Тапсырыс процесі

```
1. Үлгі таңдау     →  Мерекеге лайық шаблон
2. Дерек енгізу    →  Есімдер, күн, уақыт, мекенжай
3. Музыка + фото   →  Дайын немесе өз файлдары
4. Төлем           →  Менеджер арқылы растау
5. Жеке сілтеме    →  Қонақтарға жіберу
```

Тапсырыс беруде екі режим бар:
- **Өзім жасаймын** — барлық деректерді форма арқылы өзі толтырады, алдын ала қарайды
- **Менеджерге өткізу** — WhatsApp арқылы менеджер жасайды

## ⚙️ Орнату

### Талаптар

- PHP 8.0+
- Apache / Nginx (`.htaccess` қолдауымен)

### Жергілікті іске қосу

**Windows:**
```bat
start-server.bat
```

**Linux / macOS:**
```bash
bash start-server.sh
```

### Конфигурация

`api/config.php` файлында:

```php
const ADMIN_USERNAME = 'admin';
const ADMIN_PASSWORD_HASH = '...'; // SHA-256 хэш
```

### Баға өзгерту

`data/pricing.json` файлын редакциялаңыз:

```json
{
  "web":    { "current": 990,  "old": 1990 },
  "vipWeb": { "current": 1190, "old": 4990 }
}
```
